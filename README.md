INSTALLATION
------------
1. Installer MySQL et creer la base : importer bibliotheque_municipale.sql
2. Verifier DB.java : url = jdbc:mysql://127.0.0.1:3306/bibliotheque_municipale, user = root, mdp = ""
3. Lancer : mvn clean javafx:run
